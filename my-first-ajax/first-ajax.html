jQuery.ajax()

The syntax is jQuery.ajax( url [, settings ] ). Remember, the jqXHR objects returned by $.ajax() implements the Promise interface, giving it all the properties, methods, and behavior of a Promise (see Deferred object tab for more information).

Perform an asynchronous HTTP (Ajax) request.

To make things easier and also because doing it this way we can reach out to external servers, we are going to perform our first request to an external API, in this case the The New York Times API, although there are many more public API's you may want to use. Here is a cool link to search for API's

If you look at the syntax above, for the request we need a URL, which is a string, and for settings Plain Object, a set of key/value pairs that configure the Ajax request..

Since we did not built this API we must find out first from the creators of the NYT API what type of data they thay want. Open Developer New York Times page.

One of the requirements to be able to use this API (an for many other API's, is to obtain a Key). We will need it as part of the Ajax URL

Ajax URL

It is a String that contains the info we want from the API. We must build the URL according to API specification so we the server can correctly interpret the information we want.

Ajax Settings

Of the type PlainObject, a set of key/value pairs to configure the Ajax request. All settings are optional. Find all possible settings here
